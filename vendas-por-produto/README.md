# SQL Portfolio — BigQuery

Este repositório reúne projetos de estudo e prática em SQL utilizando o BigQuery, com foco em análise de dados e métricas de negócio.

Os projetos foram desenvolvidos com dados públicos e têm como objetivo consolidar conceitos fundamentais de SQL aplicados ao contexto de análise de dados.
